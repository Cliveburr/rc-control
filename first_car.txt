Resumo da arquitetura elétrica
- Bateria: 1S LiPo (3.7 V nominal).
- Motor: 8520 brushed coreless, controlado por um micro ESC brushed 1S (sinal tipo servo).
- ESP32: alimentado a 3.3 V por um DC-DC buck‑boost 3.3 V (1 A). GND comum com ESC e servo.
- Servo de direção:
  - Opção recomendada: micro-servo “1S compatible” (3.7 V), alimentado direto da bateria.
  - Alternativa: micro-servo 5 V + booster 5 V (caso não encontre servo 1S).

Wiring cheat-sheet (resumo)
- Battery (JST-PH) splits to:
  - ESC power input (B+/B−).
  - 3.3V buck-boost input → output to ESP32 3V3 pin.
  - If using 5V servo: battery → 5V booster → servo Vcc.
  - If using 3.7V servo: servo Vcc direto da bateria.
- Grounds: ESC GND, ESP32 GND, servo GND todos em comum.
- Signals (ESP32):
  - One GPIO (PWM/LEDC at 50 Hz) → ESC signal.
  - One GPIO (PWM/LEDC at 50 Hz) → Servo signal.
- Add 470–1000uF capacitor across ESP32 3.3V and GND near the board.

Notas rápidas
- Mantenha os fios de motor/ESC trançados e separados do 3.3V para reduzir ruído.
- Limite a tensão mínima da LiPo no seu código (ex.: pare <3.5 V/célula) e/ou use o buzzer.
- Se o diecast ficar pesado, pode trocar a relação para 6:1: “0.5M pinion 8T + spur 48T”.
